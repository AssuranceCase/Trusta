Solution(evidence): A solution provides references to evidence items. Evidence can be experimental data, historical records, analytical reports, simulation results, or other materials supporting the argument.

The five basic CAE(claim-argument-evidence) building blocks that we have identified are:
1. Decomposition: partitions some aspect of the claim.
2. Substitution: refines a claim about an object into another claim about an equivalent object.
3. Concretion: gives a more precise definition to some aspect of the claim.
4. Calculation or proof: used when some value of the claim can be computed or proved.
5. Evidence incorporation: incorporates evidence that directly supports the claim.
In practice, some of the basic blocks are often merged together into composite blocks.
